# NodPT Copilot Instructions

**NodPT** is a visual AI-assisted workflow editor built with a modern microservices architecture using Vue 3, Rete.js, Bootstrap 5, .NET 8, and Docker.

For detailed instructions, see `.github/copilot-instructions.md`.

## Quick Reference

### Project Structure

-  **Frontend/**: Vue 3 visual editor (Port 8443)
-  **WebAPI/**: .NET 8 REST API with integrated SignalR hub (Port 8846)
-  **Data/**: Shared data layer with DevExpress XPO
-  **Executor/**: Background job processor
-  **Redis/**: Message broker and caching (Port 6379)

### Key Commands

**Frontend** (in `Frontend/src`):

```bash
npm install && npm run dev     # Development
npm run build                  # Production build
npm run lint                   # Lint code
```

**WebAPI** (in `WebAPI/src`):

```bash
dotnet restore && dotnet build  # Build
dotnet run                      # Run API
```

### Important Conventions

-  Use Bootstrap 5 only for styling (no custom CSS frameworks)
-  Use the API plugin (`inject('api')`) for HTTP requests, never Axios directly
-  Never send firebaseUid to backend - JWT token handles authentication
-  Use PascalCase for data from backend, camelCase for external APIs
-  Use eventBus (Tiny Emitter) instead of watch for cross-component events
-  **Do not use C# interfaces** in service files - use concrete classes directly
